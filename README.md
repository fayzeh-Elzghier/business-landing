# Nova Studio – Business Landing Website

This project is a fully responsive multi-page business landing website built using HTML5 and CSS3.

It was created as part of the Udacity Front-End Web Developer Nanodegree program.

---

## 🚀 Project Overview

Nova Studio is a fictional business landing page designed to demonstrate:

- Semantic HTML5 structure
- Clean CSS organization
- Responsive web design
- Flexbox layout
- CSS Grid layout
- Media queries
- Accessible form structure
